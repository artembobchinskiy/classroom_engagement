# Classroom Engagement  
_Система оценки вовлеченности обучающихся по фотографиям учебных аудиторий_

## О проекте

Проект представляет собой опытный образец системы компьютерного зрения, которая по **статическим изображениям учебных аудиторий** во время занятий автоматически оценивает:

- посещаемость;
- вовлеченность обучающихся в образовательный процесс.

Система разрабатывается в рамках учебного задания  
«Разработка программной реализации модели искусственного интеллекта как объекта техники и технологий»  
по треку «Технологии искусственного интеллекта и их применение» курса  
**«Искусственный интеллект в высшей школе»**.

Проект ориентирован на потребности образовательных организаций в объективном, автоматизированном мониторинге качества преподавания и включенности студентов.

---

## Цель

**Повысить эффективность управления образовательным процессом и качество преподавания**  
за счёт создания инструмента автоматического мониторинга посещаемости и вовлеченности студентов  
на основе анализа фотографий учебных аудиторий.

Система призвана:

- снизить долю субъективных оценок посещаемости и активности;
- обеспечить регулярный мониторинг ключевых параметров учебного процесса;
- предоставить преподавателям и администрации аналитические данные для принятия решений.

---

## Основные задачи

В рамках проекта планируется:

1. **Сформировать верифицированный датасет** изображений учебных аудиторий  
   с разметкой:
   - количества обучающихся;
   - уровней их вовлеченности.

2. **Определить визуальные признаки и поведенческие индикаторы**,  
   которые коррелируют с высокой и низкой вовлечённостью (поза, направление взгляда, взаимодействие с материалами и т.п.).

3. **Разработать прототип системы компьютерного зрения**, реализующий:
   - детекцию обучающихся на изображении;
   - классификацию по уровням вовлеченности.

4. **Сформировать агрегированные метрики** по каждому кадру:
   - общее количество студентов;
   - число вовлечённых;
   - число невовлечённых;
   - доля вовлечённых.

5. **Подготовить методические рекомендации** по применению системы  
   в образовательной аналитике и управлении качеством обучения.

---

## Функциональные возможности прототипа

На уровне опытного образца система должна уметь:

- Принимать на вход статическое изображение аудитории (фотографию/кадр).
- Определять **количество обучающихся в помещении** (метрика посещаемости).
- Классифицировать каждого студента по признакам **«вовлечён / не вовлечён»**.
- Формировать выходные данные в виде:
  - таблицы или json с разметкой по каждому объекту;
  - сводных показателей по кадру (общее число, вовлечённые, невовлечённые, доля вовлечённых).

---

## Область применения

Система ориентирована преимущественно на **высшее образование** и может использоваться для:

- регулярного мониторинга посещаемости и активности на занятиях;
- оценки вовлечённости студентов в образовательный процесс;
- поддержки управленческих решений по оптимизации:
  - содержания дисциплин;
  - структуры занятий;
  - методик и стиля преподавания;
- формирования аналитических сводок на уровне:
  - кафедр;
  - факультетов;
  - образовательных программ.

---

## Актуальность

Проект соответствует задачам, обозначенным в  
**«Стратегии цифровой трансформации отрасли науки и высшего образования» Минобрнауки России**,  
и отвечает растущему запросу вузов на:

- объективные данные о посещаемости и вовлечённости;
- автоматизацию рутинного мониторинга учебного процесса;
- аналитические инструменты для оценки качества преподавания.
